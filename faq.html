Frequently Asked Questions

Q. Registration can be done through app?

A. No user can only be added through admin, and details will be provided to user for login into app.

Q. What Sort of information is need to be provide for creating a user?

A. First Name, Last Name, Email ID needs to be provided.Login details will be forwarded through admin on provided Email ID as it’s been registered.

Q. How to save scanned data for particular store/warehouse?

A. During scanning there will be an option for select Warehouse/Store. From there you can select your store and after scanning you can save data, the data will be stored for selected store.

Q. What if barcode doesn’t get scanned?

A. you can manually enter the details for that barcode.

Q. How to delete the data?

A. There will be a cross mark “x” on scan list. Click there to delete the data.

Q. Is warehouse/Store can be added from user end?

A. No, they will be added only through admin.
